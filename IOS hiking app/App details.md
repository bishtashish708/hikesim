HikeSim iOS App - App Details

Purpose
- Mobile prototype for HikeSim with an iOS-first UI.
- Mirrors the web hero + trending sections as a starting point.

Current State
- Single-screen UI in App.js.
- Static trending cards (placeholder data).
- Styled to match the web visual language.

Next Steps
- Hook the cards to the web API (trending + hike list).
- Add authentication flow and hike detail screens.
- Share plan generation logic via API calls.

Run Locally
- cd "IOS hiking app"
- npm install
- npx expo start
